# 邮箱验证码功能 - 前端集成指南

## 概述

本文档描述邮箱验证码功能的接口规范，供前端开发人员集成使用。

---

## 1. 发送验证码

### 接口信息

| 项目 | 内容 |
|------|------|
| 方法 | `POST` |
| 路径 | `/api/v1/auth/verify/send` |
| 需要认证 | 否 |

### 请求体

```json
{
  "email": "user@example.com",
  "scene": "register" | "login" | "reset_password"
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| email | string | 是 | 目标邮箱地址 |
| scene | string | 是 | 场景：`register` / `login` / `reset_password` |

### 响应 (200 OK)

```json
{
  "message": "验证码已发送至邮箱",
  "expires_in": 600
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| message | string | 提示信息 |
| expires_in | integer | 验证码有效期（秒） |

### 错误响应

| 状态码 | 错误信息 | 说明 |
|--------|----------|------|
| 400 | 该邮箱已注册，请直接登录 | register 场景邮箱已存在 |
| 400 | 该邮箱尚未注册 | login/reset_password 场景邮箱不存在 |
| 429 | 发送过于频繁，请 60 秒后重试 | 频率限制 |
| 429 | 今日发送次数已用完，请明天再试 | 超过每日次数限制 |

## 2. 验证验证码

### 接口信息

| 项目 | 内容 |
|------|------|
| 方法 | `POST` |
| 路径 | `/api/v1/auth/verify/check` |
| 需要认证 | 否 |

### 请求体

```json
{
  "email": "user@example.com",
  "scene": "register" | "login" | "reset_password",
  "code": "123456"
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| email | string | 是 | 目标邮箱地址 |
| scene | string | 是 | 场景 |
| code | string | 是 | 6位验证码 |

### 响应 (200 OK)

```json
{
  "valid": true,
  "token": "eyJhbGciOiJIUzI1NiIs..."
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| valid | boolean | 验证是否成功 |
| token | string | 验证成功时返回临时 Token（用于完成注册/重置密码） |

### 错误响应

| 状态码 | 错误信息 | 说明 |
|--------|----------|------|
| 400 | 验证码已过期，请重新发送 | 验证码超时 |
| 400 | 验证码错误，剩余 X 次尝试机会 | 验证码错误 |
| 400 | 验证失败次数过多，请重新发送验证码 | 错误5次后验证码失效 |

---

## 3. 完成注册（验证码模式）

### 接口信息

| 项目 | 内容 |
|------|------|
| 方法 | `POST` |
| 路径 | `/api/v1/auth/register/complete` |
| 需要认证 | 否 |

### 请求体

```json
{
  "verify_token": "eyJhbGciOiJIUzI1NiIs...",
  "password": "12345678",
  "username": "用户名",
  "age_verified": true,
  "terms_accepted": true,
  "privacy_accepted": true,
  "invite_code": "ABC123"
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| verify_token | string | 是 | 验证码验证成功后获得的临时 Token |
| password | string | 是 | 密码，至少8位 |
| username | string | 否 | 用户名 |
| age_verified | boolean | 是 | 年龄验证（COPPA合规，13岁以上） |
| terms_accepted | boolean | 是 | 同意用户协议 |
| privacy_accepted | boolean | 是 | 同意隐私政策 |
| invite_code | string | 否 | 邀请码 |

### 响应 (201 Created)

```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "refresh_token": "eyJhbGciOiJIUzI1NiIs...",
  "token_type": "bearer"
}
```

### 错误响应

| 状态码 | 错误信息 | 说明 |
|--------|----------|------|
| 400 | 无效的验证令牌 | Token 无效或已过期 |
| 400 | 该邮箱已注册 | 邮箱已被注册 |
| 400 | COPPA 合规年龄验证失败 | 年龄不符合要求 |

## 8. 业务流程

### 8.1 验证码注册流程

```
1. 用户输入邮箱 → POST /verify/send (scene: register)
2. 用户收取邮件，获取验证码 → POST /verify/check (scene: register, code: xxx)
3. 验证成功，获取 verify_token → POST /register/complete (verify_token: xxx, password: xxx)
4. 注册成功，获取 JWT Token
```

### 8.2 验证码登录流程

```
1. 用户输入邮箱 → POST /login/code (email: xxx)
2. 用户收取邮件，获取验证码 → POST /login/verify (email: xxx, code: xxx)
3. 登录成功，直接获取 JWT Token
```

### 8.3 忘记密码流程

```
1. 用户输入邮箱 → POST /password/reset/send (email: xxx)
2. 用户收取邮件，获取验证码 → POST /password/reset/confirm (verify_token: xxx, new_password: xxx)
3. 密码重置成功，获取 JWT Token
```

## 10. 安全限制说明

| 限制项 | 数值 | 说明 |
|--------|------|------|
| 同一邮箱同一场景发送间隔 | 60 秒 | 避免频繁发送 |
| 同一邮箱每日发送上限 | 10 次 | 防止滥用 |
| 验证码有效期 | 10/5/15 分钟 | register/login/reset_password 各场景不同 |
| 验证码输入错误上限 | 5 次 | 错误5次后验证码失效 |
| verify_token 有效期 | 15 分钟 | 用于完成注册/重置密码 |

## 11. 示例代码

### 发送验证码

```javascript
async function sendVerificationCode(email, scene) {
  const response = await fetch('/api/v1/auth/verify/send', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ email, scene }),
  });

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.detail);
  }

  return response.json();
}
```

### 验证验证码并完成注册

```javascript
async function verifyAndRegister(email, code, password, username, inviteCode) {
  // 1. 验证验证码
  const verifyResponse = await fetch('/api/v1/auth/verify/check', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ email, scene: 'register', code }),
  });

  if (!verifyResponse.ok) {
    const error = await verifyResponse.json();
    throw new Error(error.detail);
  }

  const { token: verifyToken } = await verifyResponse.json();

  // 2. 完成注册
  const registerResponse = await fetch('/api/v1/auth/register/complete', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      verify_token: verifyToken,
      password,
      username,
      age_verified: true,
      terms_accepted: true,
      privacy_accepted: true,
      invite_code: inviteCode,
    }),
  });

  if (!registerResponse.ok) {
    const error = await registerResponse.json();
    throw new Error(error.detail);
  }

  return registerResponse.json();
}
```
